# Code of Conduct – SmarterLogicWeb

Ce projet adopte le [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Notre engagement
Nous nous engageons à rendre la participation à ce projet une expérience ouverte, accueillante et respectueuse.

## Standards attendus
- Utiliser un langage ouvert et inclusif
- Respecter des points de vue différents
- Accepter les critiques constructives
- Se concentrer sur ce qui est le mieux pour la communauté et le projet

## Comportements inacceptables
- Harcèlement de toute forme
- Commentaires offensants (publics ou privés)
- Publication d’informations privées d’autrui
- Toute conduite non professionnelle

## Applicabilité
Ce code s’applique dans les espaces du projet (issues, PRs, docs, discussions) et lorsque vous représentez le projet.

## Signalement

Toutes les plaintes seront examinées et feront l’objet d’une réponse appropriée.
