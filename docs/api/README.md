# API

Ce template est principalement frontend et ne fournit pas d’API backend par défaut.

## Si vous ajoutez un backend
- Documentez vos endpoints ici (OpenAPI/Swagger recommandé).
- Exemples:
  - POST /contact – envoi de formulaire
  - GET /services – liste des services
  - GET /projects – liste des réalisations

## Conseils
- Authentification (JWT/OAuth2) si nécessaire
- Rate‑limiting et protections CSRF/CORS
